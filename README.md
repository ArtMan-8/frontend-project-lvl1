## Hexlet project 
### CLI App - math mini-games

---

## Setup
### Installing dependencies 

```bash
$ make install
```

### Installing games
#### in Linux

```bash
$ make game-linux
```

#### in Windows

```bash
$ make game-windows
```

### UnInstalling games 

```bash
$ make game-uninstall
```

[![asciicast](https://asciinema.org/a/RwUZFdUAPwTsKTmhcg4Zqog26.svg)](https://asciinema.org/a/RwUZFdUAPwTsKTmhcg4Zqog26)

## Run games

```bash
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-progression
$ brain-prime
```
